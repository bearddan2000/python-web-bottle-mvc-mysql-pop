# python-web-bottle-mvc-mysql-pop

## Description
Simple web app that serves static pages
for a bottle project.

Uses sqlalchemy query a table `pop`.

## Tech stack
- python
  - bottle
  - sqlalchemy
- bootstrap
- jquery
- dataTable
- mariadb

## Docker stack
- python:latest
- mariadb:latest

## To run
`sudo ./install.sh -u`
- [Availble at](http://localhost)

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
